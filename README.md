# Aplikasi Perencanaan Desa

Dashboard monitoring kebun organik komunitas yang terinspirasi dari template Organic ThemeWagon.

## Cara membuka

Jalankan dengan PHP built-in server:

```powershell
cd D:\rpjrkp
php -S localhost:8000
```

Lalu buka `http://localhost:8000/`.

## Fitur

- Header dan visual hijau-kuning bertema Organic
- Metrik panen, kegiatan, relawan, dan distribusi komunitas
- Grafik hasil panen dan status distribusi
- Filter komoditas, pencatatan kegiatan, pencarian kegiatan, tabel aktivitas, sumber daya kritis, area kegiatan, dan agenda kebun
